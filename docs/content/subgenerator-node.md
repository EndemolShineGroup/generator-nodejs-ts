## Usage

```bash
yo @endemolshinegroup/nodejs-ts:node
```

### Options

#### `projectScope`

- Required: Yes
- Type: String

#### `projectName`

- Required: Yes
- Type: String

#### `projectDescription`

- Required: Yes
- Type: String

#### `version`

- Required: Yes
- Type: String

#### `isPublic`

- Required: Yes
- Type: Boolean
